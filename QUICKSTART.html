üéØ How to Use

Adding a Script

  1. Click "Add New Script" button
  2. Fill in:
    • Script Name (e.g., "Read PLC Tag Value")
    • Category (PLC Comms, Calculations, Alarms, or Data)
    • Description (what it does)
    • Tags (comma-separated keywords)
    • Your QuickScript/VBScript code
  3. Click "Save Script"

Using Parameter Placeholders

Use {{PARAMETER_NAME}} for values that change:

' Example with parameters
Dim tagValue
tagValue = ReadValue({{TagName}})
{{ResultTag}} = tagValue

When using the script, replace:

  • {{TagName}} ‚Üí "Reactor1.Temperature"
  • {{ResultTag}} ‚Üí "Display_Temp"

Finding Scripts

  • Search Box: Type keywords to search across all fields
  • Category Filters: Click categories in sidebar
  • Sort: By name, date, or category

Exporting Scripts

Single Script:

  1. Click script card to view details
  2. Click "Export .txt" or "Export .script"
  3. File downloads with header comments

Entire Library:

  1. Click export icon in header
  2. JSON file downloads (backup)
  3. Use "Import" to restore later

üîß Common Use Cases

1. Tag Reading with Error Handling

' Read PLC Tag Value with Error Handling
If InTouchTag({{TagName}}) Then
    {{ResultTag}} = ReadValue({{TagName}})
Else
    LogMessage "Error: Tag not found"
    {{ResultTag}} = 0
End If

2. Batch Alarm Acknowledgment

' Acknowledge All Active Alarms
alarmCount = wwAlarmTotalRecords()
If alarmCount > 0 Then
    result = wwAcknowledgeAllAlarms()
End If

3. String Parsing (Barcodes)

' Parse Delimited String
parts = Split({{InputString}}, "{{Delimiter}}")
{{Part1Tag}} = Trim(parts(0))
{{Part2Tag}} = Trim(parts(1))
